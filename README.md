# 📄 Resume Builder

A modern and responsive Resume Builder web application created using **HTML, CSS, and JavaScript**.
This project helps users create a professional resume by filling out simple forms step-by-step and instantly previewing the final resume.

---

## 🚀 Features

* ✅ Personal Details Section
* ✅ Education Details
* ✅ Experience Section
* ✅ Skills Section
* ✅ Projects Section
* ✅ Live Resume Preview
* ✅ Edit Any Section Anytime
* ✅ Save / Print Resume
* ✅ Responsive Layout
* ✅ Clean User Interface

## ⚙️ How the Project Works

1. Enter Personal Details
2. Add Education Information
3. Add Experience Details
4. Add Skills
5. Add Project Information
6. Click on **Finish**
7. Preview Resume Instantly
8. Edit Any Section if Needed
9. Click **Save Resume** to Print or Download
